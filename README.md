# 🧩 AutiCare – A Supportive Mobile Application for Children with Autism Spectrum Disorder (ASD)

## 📖 Overview

**AutiCare** is an Android application developed to support children with **Autism Spectrum Disorder (ASD)** in their daily communication, learning, emotional development, and routine management. The application provides an easy-to-use, child-friendly interface with visual communication tools, emotion tracking, and caregiver monitoring features.

The project aims to improve communication between children, parents, and caregivers while encouraging independent learning through interactive activities.

---

## 🎯 Project Objectives

- Improve communication for children with ASD.
- Support emotional recognition and development.
- Help children follow daily routines.
- Allow parents and caregivers to monitor progress.
- Provide a simple and sensory-friendly user interface.
